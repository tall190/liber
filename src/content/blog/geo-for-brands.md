마케터라면 한 번쯤 이런 경험이 있을 거예요.

ChatGPT나 Claude에 "OO 카테고리 브랜드 추천해줘"라고 물어봤는데, 경쟁사 이름은 나오는데 우리 브랜드는 안 나오는 거죠. 분명히 우리가 더 좋은 제품인데.

이게 기분 나쁜 문제만이 아니에요. 앞으로 이런 일이 훨씬 더 자주 일어날 거예요.

---

## 사람들이 AI에게 묻기 시작했다

브랜드를 알리는 방법은 시대마다 달랐어요.

신문 광고 → TV 광고 → 검색 광고 → SNS 광고. 채널이 바뀔 때마다 잘 적응한 브랜드가 앞서 나갔죠.

지금 다음 채널이 열리고 있어요.

사람들이 AI에게 자연어로 물어보기 시작했거든요.

> "강남에서 분위기 좋은 이탈리안 레스토랑 추천해줘."
> "30대 여성 선물로 좋은 향수 브랜드 알려줘."
> "소규모 쇼핑몰 마케팅 자동화 솔루션 비교해줘."

검색창에 키워드를 입력하는 대신, AI에게 맥락까지 담아서 물어봐요. AI는 링크 목록 대신 문장으로 답해요. 특정 브랜드를 직접 언급하면서, 왜 그 브랜드가 적합한지까지 설명해요.

이 순간이 새로운 홍보의 접점이에요.

![마케팅 채널의 역사적 흐름 — 신문 광고에서 AI 추천까지](../../assets/ai-ready-ui_ai_recommand_image.png)

## AI 추천이 광고와 다른 이유

검색 광고는 "클릭"을 샀어요. 내 광고가 상단에 뜨고, 사용자가 클릭하면 비용이 나가요. CPC(클릭당 비용) 모델.

AI 추천은 달라요.

AI가 "OO 브랜드가 이 상황에 가장 적합해요"라고 말하면, 사용자는 광고로 인식하지 않아요. 지인이 추천해주는 것처럼 받아들이죠. 신뢰도가 다른 거예요.

그리고 지금은 돈을 내고 살 수 없어요. AI 추천은 AI가 브랜드를 이해하고 있어야 가능해요. 이해하지 못하면 추천 자체가 없는 거예요.

---

## 문제: 대부분의 브랜드 페이지는 AI가 이해 못한다

솔직하게 봐봅시다. 대부분의 브랜드 홈페이지는 이렇게 생겼어요.

**일반적인 홈페이지 (AI가 이해하기 어려운 구조)**

```
[큰 배경 이미지]

당신의 일상을 특별하게.

[버튼: 더 알아보기]
```

아름답죠. 하지만 AI는 이 페이지에서 이 브랜드가 무엇을 하는 회사인지, 누구를 위한 서비스인지, 경쟁사와 무엇이 다른지 파악하지 못해요.

상품 페이지도 비슷해요.

```
[상품 이미지 6장]

프리미엄 보습 크림

최고급 성분으로 만든 당신의 피부를 위한 선택.

₩45,000
```

"최고급 성분"이 뭔지, "당신의 피부"가 어떤 타입인지, 왜 ₩45,000이 합리적인지 — AI는 모르는 거예요.

**AI-ready로 바꾼 버전**

```
[상품 이미지]

히알루론산 보습 크림 50ml

- 피부 타입: 건성, 복합성 피부 적합
- 주요 성분: 히알루론산 3중 구조 (분자량 다층), 세라마이드 EOP
- 수분 유지 시간: 임상 테스트 기준 72시간
- 무향, 무색소, 알레르기 테스트 완료
- 피부과 전문의 임상 검증
- 소비자 만족도: 92% (2024년 구매자 1,200명 기준)

₩45,000 / 50ml (100ml 기준 ₩90,000)
```

감성 카피가 사라지고 사실이 들어왔어요. AI는 이 상품을 이해하고, 적합한 질문이 들어오면 인용할 수 있어요.

## AI-ready Marketing: 어떻게 준비하나요?

AI가 브랜드를 이해하고 추천할 수 있게 만드는 것 — 이걸 **GEO(Generative Engine Optimization)** 라고 불러요. SEO가 검색 엔진 크롤러를 위한 최적화였다면, GEO는 AI 언어 모델을 위한 최적화예요.

### 1. 브랜드 핵심 정보를 명확하게 구조화하기

홈페이지 어딘가에 이런 내용이 텍스트로 있어야 해요.

- 우리는 무엇을 하는 회사인가 (한 문장으로)
- 누구를 위한 서비스인가 (타깃 고객)
- 경쟁사와 무엇이 다른가 (핵심 차별점)
- 실적과 신뢰를 보여주는 수치

슬로건이 아닌 사실로. 감성이 아닌 정보로.

### 2. 상품/서비스 페이지에 구체적 사실 채우기

"최고의 품질"은 AI가 인용하지 않아요. "3년 연속 소비자만족도 1위"는 인용해요.

구체적인 수치, 검증 가능한 사실, 비교 가능한 정보 — 이런 것들이 AI의 인용 대상이 돼요.

### 3. FAQ 페이지를 실제 질문-답변으로 만들기

AI는 사용자의 질문에 답하기 위해 웹을 탐색해요. FAQ 페이지가 실제 질문 형태로 구성되어 있으면, AI가 해당 내용을 그대로 인용할 수 있어요.

❌ 일반적인 FAQ:
```
Q. 배송은 얼마나 걸리나요?
A. 평균 3-5일 소요됩니다.
```

✅ AI-ready FAQ:
```
Q. OO 크림이 민감한 피부에도 사용할 수 있나요?
A. 네, OO 크림은 피부과 전문의 임상 테스트를 완료했고,
   무향·무색소 포뮬라로 민감성 피부에도 사용 가능합니다.
   다만 심한 아토피나 스테로이드 치료 중인 경우에는
   사용 전 전문의 상담을 권장드립니다.
```

"OO 크림이 민감한 피부에 괜찮아요?" 라는 질문에 AI가 이 답변을 그대로 인용할 수 있어요.

### 4. 구조화된 데이터(Schema.org) 적용하기

웹페이지 HTML에 Schema.org 마크업을 추가하면, AI가 콘텐츠 구조를 더 정확하게 파악해요. 기술적인 내용이지만, 개발팀에 요청하면 쉽게 적용할 수 있어요.

**상품 페이지 예시 (JSON-LD):**

```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Product",
  "name": "히알루론산 보습 크림 50ml",
  "description": "히알루론산 3중 구조와 세라마이드 EOP로 72시간 수분 유지. 건성·복합성 피부 적합.",
  "brand": {
    "@type": "Brand",
    "name": "OO 브랜드"
  },
  "aggregateRating": {
    "@type": "AggregateRating",
    "ratingValue": "4.7",
    "reviewCount": "1200"
  },
  "offers": {
    "@type": "Offer",
    "price": "45000",
    "priceCurrency": "KRW",
    "availability": "https://schema.org/InStock"
  }
}
</script>
```

**브랜드 홈페이지 예시 (JSON-LD):**

```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "OO 브랜드",
  "description": "2018년 설립, 민감성 피부를 위한 더마 코스메틱 브랜드. 피부과 전문의와 공동 개발.",
  "foundingDate": "2018",
  "numberOfEmployees": "50",
  "sameAs": [
    "https://www.instagram.com/oo_brand",
    "https://blog.naver.com/oo_brand"
  ]
}
</script>
```

AI는 이 구조화 데이터를 직접 파싱해요. 브랜드에 대한 사실을 명확하게 전달할 수 있어요.

### 5. 외부에서 언급되는 환경 만들기

AI는 여러 출처에서 반복적으로 언급되는 브랜드를 더 신뢰해요. 자사 홈페이지만으로는 부족해요.

업계 미디어 기고, 전문가 리뷰, 비교 분석 콘텐츠, 커뮤니티 언급 — 이런 외부 언급이 AI의 신뢰도 평가에 영향을 줘요.

[Profound의 GEO 연구](https://www.tryprofound.com/resources/articles/generative-engine-optimization-geo-guide-2025)에 따르면, AI 인용 출처의 **21.74%가 Reddit·YouTube 같은 UGC 플랫폼**에서 나와요. 자사 사이트보다 커뮤니티의 진짜 사용자 경험이 AI의 판단에 더 큰 영향을 주는 거예요.

## AI 추천이 가져오는 홍보 효과

준비된 브랜드가 얻는 게 세 가지예요.

**광고비 없는 노출.** AI 추천은 현재 유료 채널이 없어요. AI가 브랜드를 이해하면, 관련 질문이 들어올 때마다 추천 후보에 올라요. 키워드 경쟁 없이, 클릭 비용 없이.

**신뢰도 높은 도달.** 광고를 클릭한 사람보다 AI 추천을 따라온 사람의 구매 의도가 높아요. "AI가 추천했다"는 게 이미 하나의 필터링이거든요.

**선점 효과.** AI가 특정 카테고리에서 특정 브랜드를 자주 추천하면, 그 이력이 쌓여서 더 자주 추천돼요. 먼저 시작하는 게 유리해요. [Strapi의 2025년 상반기 분석](https://strapi.io/blog/generative-engine-optimization-geo-guide)에 따르면, AI 검색에서 유입된 세션이 전년 대비 **527% 증가**했어요. 이 채널은 이미 실재하고 있어요.

---

브랜드를 알리는 채널은 계속 바뀌어왔어요. 이번에도 바뀌고 있고요. 검색 광고에 최적화했던 것처럼, 이제는 AI에게 이해받을 차례예요.

AI가 브랜드를 이해해야, 추천할 수 있으니까요.
